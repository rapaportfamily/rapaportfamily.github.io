The JewishGen Holocaust Database

JewishGen's Holocaust Database is a unique and critically valuable collection of databases which contain information about Holocaust victims and survivors.  As of May 2025, more than 6 million records are available in this continually updated collection, which includes Camp Records, Transport Lists, Name Lists, and a multitude of of other record types.

This is an ongoing project, and we welcome additional volunteers.

Search Hint!

(1) Search by OR: You can uses commas to search for either of two names. For example, COHEN or GOLDBERG should be listed as COHEN,GOLDBERG.
(2) Search by AND: You can use a space to search for two names at the same time. For example, COHEN and GOLDBERG should be listed as COHEN GOLDBERG.
(3) Exact search: will include Jewishgen synonyms for town and given names. For example, a search for Lodz will also return records that include the name Litzmannstadt and an exact search for the first name Minnie will include the names Miriam, Miryam, Mirjam, Maria, Mariam, Marjam, Maryam, Marja, Minnie, Minie, Mary, Marie, Mina, Minna, or Mini
